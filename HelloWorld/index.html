Warum solltest Du JavaScript lernen?

  • Aufpeppen einer serverseitig gerenderten Anwendung
  • Integration von JS-Bibiliotheken mit GWT, Vaadin oder JBoss Errai
  • HTML5 Entwicklung am effektivsten mit JavaScript
  • Die Sprache JavaScript enthält interessante Konzepte
  • Für Entwickler: Wer JavaScript und Java kann hat einen Wettbewerbsvorteil

Falsche Freunde für Java-Entwickler

  • Nur die Syntax ist ähnlich wie Java, die Semantik unterschiedlich
  • this kann an so manches gebunden sein
  • Google ist NICHT dein Freund
  • Copy-Paste-Based vs. OO-Based
  • HTML/CSS/JavaScript

Mehr Stolpersteine

  • Felder werden nicht deklariert
  • Objekte sind wie Maps
  • Keine statischen Typen
  • Blöcke spannen keine Scopes auf
